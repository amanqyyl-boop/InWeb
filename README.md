# 📚 湾区书屋 — 大湾区大学（GBU）阅读平台

> 基于 Flask + 原生前端的智能小说阅读与推荐系统

---

## 📖 项目简介

**湾区书屋** 是一个面向大湾区大学（GBU）师生的在线阅读平台，集 **小说浏览、分类检索、智能推荐、在线阅读、书架管理、读者评论** 于一体。

项目以本地 `小说库/` 目录中的真实 txt 文件为数据源，自动扫描入库、解析章节、提取简介，无需手动维护数据库。后端采用 **1D CNN 深度学习模型** 实现个性化推荐，用户越用越"懂"你。

## 🏗 技术实现

```
InWeb/
├── frontend/                  # 前端
│   ├── index.html             # 页面结构（SPA 单页应用）
│   ├── style.css              # 样式（亮色/暗色主题）
│   └── app.js                 # 前端逻辑（SPA 路由、API 调用、状态管理）
├── backend/                   # 后端
│   ├── app.py                 # Flask 服务器（API + 静态文件服务）
│   ├── cnn_recommender.py     # 1D CNN 推荐引擎（纯 NumPy 实现）
│   └── accounts/              # 用户数据存储（文件系统）
├── 小说库/                    # 小说数据源
│   ├── 玄幻类/斗破苍穹/...
│   ├── 仙侠类/圣墟/...
│   └── ...
└── README.md
```

### 前端技术

- **纯原生 HTML + CSS + JavaScript** — 无任何前端框架依赖
- **SPA 单页架构** — 通过 `navigateTo()` 实现页面切换，保存滚动位置
- **CSS 变量主题系统** — 亮色/暗色主题一键切换
- **Font Awesome 6** — 图标库
- **响应式设计** — 适配桌面和移动端

### 后端技术

- **Flask 2.3** — Python Web 框架
- **RESTful API** — 前后端通过 JSON API 通信
- **文件系统存储** — 用户信息、书架、评论、浏览记录均以 txt 文件存储
- **自动小说扫描** — 启动时递归扫描 `小说库/` 目录，自动解析章节

### 智能推荐引擎

项目核心亮点是一个 **纯 NumPy 实现的 1D CNN（一维卷积神经网络）**：

```
用户浏览历史 → Embedding(8维) → Conv1D(12 filters, kernel=3) → ReLU → GlobalMaxPool → Dense → Tanh → 偏好得分
```

- **语义化 Embedding 初始化**：相似分类（如玄幻→仙侠）的向量更接近
- **在线增量学习**：用户每浏览一本小说，模型自动微调权重
- **综合评分公式**：

$$\text{总分} = \text{CNN 偏好分} \times 0.6 + \text{注册偏好} \times 0.25 + \text{热度/评分} \times 0.15$$

## 🎨 设计思想

### 1. 零数据库依赖
所有数据（用户、书架、评论、小说）均以 **文件系统 + txt 格式** 存储，省去数据库安装配置，开箱即用。

### 2. 前后端分离
前端通过 AJAX 调用后端 REST API，后端不渲染任何 HTML 模板，职责清晰，易于扩展。

### 3. 数据驱动
`小说库/` 目录中放置任意 txt 小说文件，重启即自动识别，**无需修改任何代码**。

### 4. 轻量级 ML 推理
推荐引擎不依赖 TensorFlow/PyTorch，仅用 NumPy 实现完整的 CNN 前向传播和反向传播，模型权重仅 ~2KB。
