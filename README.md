# 湾区书屋 — 大湾区大学（GBU）阅读平台

基于 Flask + 原生前端的智能小说阅读与推荐系统。

---

## 项目简介

湾区书屋是一个面向大湾区大学（GBU）师生的在线阅读平台，集小说浏览、分类检索、智能推荐、在线阅读、书架管理、段落评论于一体。

项目以本地 `小说库/` 目录中的真实 txt 文件为数据源，自动扫描入库、解析章节、提取简介。后端采用 scikit-learn（KMeans 聚类 + Cosine Similarity）实现个性化推荐。

---

## 核心功能

- **在线阅读器** — 从本地 txt 文件实时读取章节内容，支持上下翻页、键盘快捷键
- **段落评论** — 阅读时可在每一段后面添加评论，支持回复、赞同/否定，所有用户可见
- **章节评论** — 详情页支持对整章发表评论，同样支持回复和投票
- **搜索与筛选** — 按书名/作者搜索，按分类和排序筛选
- **智能推荐** — 基于 scikit-learn 的混合推荐（KMeans 聚类 + Cosine Similarity），根据浏览历史在线学习
- **用户系统** — 注册/登录、偏好设置、浏览记录追踪
- **书架管理** — 收藏小说，记录阅读进度
- **上传小说** — 支持拖拽上传 txt 文件，可选择"已有小说"（手动填作者）或"我的创作"（自动使用用户名）
- **夜间模式** — 一键切换护眼夜间主题
- **自动扫描** — 启动时自动扫描 `小说库/` 目录，新增小说无需改代码

## 技术实现

```
InWeb/
```

### 前端

- 纯原生 HTML + CSS + JavaScript，无框架依赖
- SPA 单页架构，页面切换无刷新
- CSS 变量主题系统，一键切换亮色/暗色
- 响应式设计，适配桌面和移动端

### 后端

- Flask 3.1 + 文件系统评论存储
- RESTful API，前后端通过 JSON 通信
- 评论系统使用 JSON 文件存储，支持多用户、回复、投票
- 用户信息、书架、浏览记录以文件系统存储
- 自动小说扫描，启动时递归扫描 `小说库/` 目录

### 推荐引擎

使用 scikit-learn 实现的混合推荐系统：

```
特征工程：分类 one-hot + 评分归一化 + 热度归一化 → 12 维特征向量
标准化：StandardScaler
聚类：KMeans（8 簇）
相似度：Cosine Similarity
评分：聚类匹配(35%) + 内容相似度(30%) + 注册偏好(25%) + 基础质量(10%)
学习：每浏览 10 本不同书后自动重聚类
```

### 评论系统

- JSON 文件存储，所有用户共享
- 支持两级结构：段落评论（阅读器中）和章节评论（详情页中）
- 每条评论支持回复无限层
- 每条评论支持赞同/否定投票
- 段落评论按钮在鼠标悬停时显示，不占用阅读空间

---

## 设计思想

1. **零数据库依赖** — 用户数据以文件系统存储，省去数据库配置
2. **前后端分离** — 前端通过 AJAX 调用 REST API
3. **数据驱动** — `小说库/` 放置 txt 文件即自动识别
4. **轻量级 ML** — scikit-learn 实现，无需 GPU，模型推理毫秒级
5. **段落级评论** — 评论附着在具体段落上，不影响阅读体验
